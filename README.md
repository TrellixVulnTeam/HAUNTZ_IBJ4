# HAUNTZ
Repositório do sistema GenesisQ

Para iniciar o sistema siga as etapas:
- Apague a pasta venv
- Criei uma nova pasta venv com o virtualenv
- Inicie o ambiente virtual através do comando -> venv/scripts/activate
- Instale as dependências através do comando -> pip install -r requirements.txt
- Rode o servidor do sistema através do comando -> python manage.py runserver
